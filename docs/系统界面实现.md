## 5.1 管理员登录

登录页面比较简单，就是邮箱输入框、密码输入框、登录按钮。背景加了点动态粒子效果，好看点。

登录的时候后端会验证邮箱密码，验证通过返回 JWT Token 和用户信息（包括角色）。前端根据角色判断跳转到哪个页面：普通用户跳首页，管理员可以进管理后台。

**核心代码（前端：登录请求 → 存 Token → 拉取资料写入本地）：**

```typescript
// frontend/src/pages/Login/index.tsx（节选）
const handleLogin = async (e: React.FormEvent) => {
  e.preventDefault();
  const data = await api.post('/user/login', formData);
  if (data.code === 200) {
    setToken(data.data);
    const userRes = await api.get('/user/profile');
    if (userRes.code === 200) setUserInfo(userRes.data);
    navigate('/', { replace: true });
    window.dispatchEvent(new Event('storage'));
  }
};
```

**核心代码（后端：校验密码并签发 JWT，载荷中含用户身份）：**

```java
// src/main/java/.../service/serviceImpl/UserServiceImpl.java（节选）
public String login(String email, String password) {
    User user = lambdaQuery().eq(User::getEmail, email).one();
    validateUserExists(user);
    if (!Objects.equals(hashPasswordWithSalt(password, user.getSalt()), user.getPassword())) {
        throw new BusinessException(ERROR_PASSWORD_WRONG);
    }
    user.setLastLoginTime(LocalDateTime.now());
    this.updateById(user);
    return JwtToken.generateToken(String.valueOf(user.getId()), user.getIdentity());
}
```

**核心代码（本地身份与管理员判断，供导航或扩展跳转使用）：**

```typescript
// frontend/src/utils/auth.ts（节选）
export const setUserInfo = (user: Partial<UserInfo>): void => {
  if (user.id) localStorage.setItem('id', user.id);
  if (user.name) localStorage.setItem('name', user.name);
  if (user.identity) localStorage.setItem('identity', user.identity);
};
export const isAdmin = (): boolean => {
  return localStorage.getItem('identity') === 'admin';
};
```

**核心代码（Python 代理：SSE 封装 + 按模型流式输出）：**

```python
# ai-proxy/main.py（节选）
def sse_event(payload: dict) -> bytes:
    return f"data: {json.dumps(payload, ensure_ascii=False)}\n\n".encode("utf-8")

@app.post("/v1/chat/stream")
async def chat_stream(request: ChatStreamRequest) -> StreamingResponse:
    provider = get_provider(request.model)
    actual_model = normalize_model_name(request.model)
    normalized = request.model_copy(update={"model": actual_model})

    async def event_stream() -> AsyncIterator[bytes]:
        yield sse_event({"type": "meta", "provider": provider.provider_name, "model": actual_model})
        async for chunk in provider.stream(normalized):
            if chunk:
                yield sse_event({"type": "content", "delta": chunk})
        yield sse_event({"type": "done"})

    return StreamingResponse(
        event_stream(),
        media_type="text/event-stream",
        headers={"Cache-Control": "no-cache", "Connection": "keep-alive", "X-Accel-Buffering": "no"},
    )
```
