常见错误

错误类型 原因 解决方案

Java 提交规范

特别注意： Java 类名必须为 Main，且不能有package声明！

基本结构

Java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        // 你的代码
        sc.close();
    }
}

常见问题

  • 类名不是Main会导致编译错误
  • 忘记关闭Scanner可能导致资源泄漏
  • 使用package声明会导致判题失败
  • Java程序启动较慢，注意时间限制

输入输出优化

C++
// 提高C++输入输出速度（大数据量时使用）
ios::sync_with_stdio(false);
cin.tie(0);

// 读取一个整数
int n;
cin >> n;

// 读取一行多个整数
int arr[100];
for (int i = 0; i < n; i++) {
    cin >> arr[i];
}

// 输出
cout << "结果: " << ans << endl;

注意事项

  • 不要使用conio.h等非标准库
  • 数组大小要足够（避免越界）
  • 变量要初始化
  • 使用endl会刷新缓冲区，大数据量时使用'\n'

通用提交规则

判题流程

  1. 提交代码到判题系统
  2. 系统编译（如需要）并执行代码
  3. 用测试用例验证输出
  4. 返回判题结果

所有语言通用规则

  • 程序必须从标准输入读取数据
  • 结果必须输出到标准输出
  • 不要尝试访问文件系统或网络
  • 不要使用危险系统调用
  • 禁止任何形式的作弊行为

结果状态说明

状态 含义
Accepted 答案正确
Wrong Answer 答案错误
Compile Error 编译失败
Time Limit Exceeded 超过时间限制
Runtime Error 运行时错误
Memory Limit Exceeded 超过内存限制
